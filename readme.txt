== Description ==

This plugin will randomly display quotes from the Stargate franchise on every page of your user dashboard. Based on the Hello Dolly plugin by Matt Mullenweg.

Tested on Multisite!

I would like to thank Seth at http://sethradio.com/ for all of his help with this plugin.

== Installation ==

= Repository: =

1. Navigate to http://YOURSITE.COM/wp-admin
2. Hover over the "Plugins" tab, and click "Add New"
3. Search for "Stargate Quotes"
4. Click "Install" then click "Ok" or "Yes" in the pop-up to confirm the install.
5. Upon installation, click "Activate"

= FTP: =
1. Download and un-zip the 'Stargate-quotes' folder
2. Upload the `stargate-quotes` folder to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Zip: =

1. Download and un-zip the 'Stargate-quotes' folder
2. RE-zip the folder
3. Navigate to `http://YOURSITE.COM/wp-admin/plugin-install.php?tab=upload`
4. Upload the re-zipped file
